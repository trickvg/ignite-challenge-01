## Getting Started

```bash
# Cloning the repository
git clone https://github.com/trickvg/ignite-challenge-01

cd ignite-challenge-01
```

### Install dependencies and running application

##### Yarn

```bash
#Installing dependencies
yarn install

# Running app
yarn dev

# Running test suit
yarn test
```
##### NPM
```bash
#Installing dependencies
npm install

# Running app
npm run dev

# Running test suit
npm test
```
